# Changelog

## v0.2.0

- Added real hybrid sample apps from source: Android WebView and iOS WKWebView.
- Added `android_hybrid_demo` and `ios_hybrid_demo` capability profiles.
- Added `--hybrid-example` CLI shortcut and a runnable hybrid Appium test.
- Added manual self-hosted/device-farm workflow for real device examples.
- Added example docs, device CI docs, documentation screenshots, and a starter project template.
- Marked the GitHub repository as usable as a template repository.

## v0.1.0

- Initial standalone mobile automation framework.
- Added Android and iOS native examples using Appium and TheApp.
- Added Appium mobile web example for Android Chrome and iOS Safari profiles.
- Added Screen Object Model, reusable flows, capability profiles, reports, artifacts, and helper catalog.
- Added action-level retries, text-entry verification, iOS keyboard typing helper, and conservative self-healing locators.
- Added helper/unit tests, GitHub Actions validation, and release-ready documentation.
